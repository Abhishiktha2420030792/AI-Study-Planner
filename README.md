# AI Study Planner

AI Study Planner is an Agentic AI web application built using Flask and IBM watsonx.ai.

## Features

- Personalized Study Planner
- Subject Prioritization
- Revision Planner
- Productivity Tips
- Motivation

## Technologies

- Python
- Flask
- IBM watsonx.ai
- IBM Cloud
- Meta Llama / IBM Granite
- HTML
- CSS
- Bootstrap
